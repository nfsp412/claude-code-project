### 1.1 项目简介

JSON格式化工具是一个纯前端工具类页面，提供JSON数据的格式化、压缩、验证和复制功能。

### 1.2 架构说明

- **前端**: Vue 3 + TypeScript
- **状态管理**: 组件内部状态
- **数据处理**: 完全在前端使用原生JSON API处理
- **API支持**: 当前无后端API，预留扩展接口

## 2. 数据结构定义

### 2.1 AppState（应用状态）

```typescript
interface AppState {
  input: string;           // 用户输入的JSON字符串
```

```typescript
interface OutputData {
```

```typescript
interface Notification {
```

## 4. 事件接口定义

### 4.1 FormatEvent（格式化事件）

```typescript
interface FormatEvent {
  type: 'format';
  payload: {
    input: string;              // 输入的JSON字符串
    indent?: number;            // 缩进空格数，默认2
  };
}
```

### 4.2 CompressEvent（压缩事件）

```typescript
interface CompressEvent {
  type: 'compress';
  payload: {
    input: string;              // 输入的JSON字符串
  };
}
```

### 4.3 ClearEvent（清空事件）

```typescript
interface ClearEvent {
  type: 'clear';
  payload: {
    target?: 'input' | 'output' | 'all';  // 清空目标，默认all
  };
}
```

### 4.4 CopyEvent（复制事件）

```typescript
interface CopyEvent {
  type: 'copy';
  payload: {
    source: 'input' | 'output';           // 复制源
    text: string;                         // 要复制的文本
  };
}
```

## 5. 类型定义（TypeScript）

```typescript
// ====== 基础类型 ======
type JsonValue = string | number | boolean | null | JsonObject | JsonArray;

interface JsonObject {
  [key: string]: JsonValue;
}

type JsonArray = JsonValue[];

// ====== 应用状态类型 ======
interface AppState {
  input: string;
  output: string;
  isValid: boolean;
  errorMessage: string;
}

// ====== 组件Props类型 ======
interface JsonFormatterProps {
  initialState?: Partial<AppState>;
  onStateChange?: (state: AppState) => void;
  maxInputSize?: number;
  theme?: 'light' | 'dark';
}

// ====== 通知类型 ======
type NotificationType = 'success' | 'error' | 'info';

interface Notification {
  type: NotificationType;
  message: string;
  duration?: number;
}

// ====== 事件类型 ======
type AppEvent = FormatEvent | CompressEvent | ClearEvent | CopyEvent;

interface FormatEvent {
  type: 'format';
  payload: {
    input: string;
    indent?: number;
  };
}

interface CompressEvent {
  type: 'compress';
  payload: {
    input: string;
  };
}

interface ClearEvent {
  type: 'clear';
  payload: {
    target?: 'input' | 'output' | 'all';
  };
}

interface CopyEvent {
  type: 'copy';
  payload: {
    source: 'input' | 'output';
    text: string;
  };
}

// ====== 结果类型 ======
type Result<T> = { success: true; data: T } | { success: false; error: string };

// ====== JSON工具函数类型 ======
type FormatResult = Result<string>;
type ValidateResult = Result<JsonValue>;
```

## 6. 后端API接口（扩展预留）

> 说明：当前版本为纯前端实现，以下接口为未来扩展预留

### 6.1 格式化接口

**接口名称**: 格式化JSON

**请求方法**: `POST`

**请求路径**: `/api/format`

**请求头**:
```
Content-Type: application/json
```

**请求参数**:
```json
{
  "json": "{\"name\":\"test\",\"value\":123}",
  "indent": 2,
  "sortKeys": false
}
```

| 参数 | 类型 | 必填 | 说明 |
|-----|------|------|------|
| json | string | 是 | 原始JSON字符串 |
| indent | number | 否 | 缩进空格数，默认2 |
| sortKeys | boolean | 否 | 是否排序key，默认false |

**响应示例**:
```json
{
  "code": 0,
  "message": "success",
  "data": {
    "formatted": "{\n  \"name\": \"test\",\n  \"value\": 123\n}",
    "originalSize": 27,
    "formattedSize": 36
  }
}
```

---

### 6.2 压缩接口

**接口名称**: 压缩JSON

**请求方法**: `POST`

**请求路径**: `/api/compress`

**请求参数**:
```json
{
  "json": "{\n  \"name\": \"test\"\n}"
}
```

| 参数 | 类型 | 必填 | 说明 |
|-----|------|------|------|
| json | string | 是 | 原始JSON字符串 |

**响应示例**:
```json
{
  "code": 0,
  "message": "success",
  "data": {
    "compressed": "{\"name\":\"test\"}",
    "originalSize": 20,
    "compressedSize": 15
  }
}
```

---

### 6.3 验证接口

**接口名称**: 验证JSON格式

**请求方法**: `POST`

**请求路径**: `/api/validate`

**请求参数**:
```json
{
  "json": "{\"name\":\"test\"}"
}
```

**响应示例**（成功）:
```json
{
  "code": 0,
  "message": "success",
  "data": {
    "valid": true,
    "parsed": {
      "name": "test"
    },
    "type": "object"
  }
}
```

**响应示例**（失败）:
```json
{
  "code": 1001,
  "message": "JSON parse error",
  "data": {
    "valid": false,
    "error": "Unexpected token } in JSON at position 5",
    "position": 5
  }
}
```

## 7. 错误码定义

### 7.1 通用错误码

| 错误码 | 说明 | HTTP状态码 |
|-------|------|-----------|
| 0 | 成功 | 200 |
| 1000 | 未知错误 | 500 |
| 1001 | 参数错误 | 400 |
| 1002 | 未授权 | 401 |
| 1003 | 禁止访问 | 403 |
| 1004 | 资源不存在 | 404 |
| 1005 | 请求方法不支持 | 405 |
| 1006 | 请求过于频繁 | 429 |
| 1007 | 服务器内部错误 | 500 |
| 1008 | 服务不可用 | 503 |

### 7.2 JSON处理错误码

| 错误码 | 说明 |
|-------|------|
| 2001 | JSON格式无效 |
| 2002 | JSON解析失败 |
| 2003 | JSON大小超限 |
| 2004 | JSON嵌套层级超限 |
| 2005 | JSON包含不支持的类型 |

### 7.3 业务错误码

| 错误码 | 说明 |
|-------|------|
| 3001 | 历史记录不存在 |
| 3002 | 分享链接已过期 |
| 3003 | 分享链接不存在 |
| 3004 | 存储空间不足 |

### 7.4 错误响应格式

## 8. 测试接口示例

### 8.1 前端组件测试用例

#### 8.1.1 格式化功能测试

```typescript
import { describe, it, expect } from 'vitest';
import { JsonFormatter } from './JsonFormatter';

describe('JsonFormatter 格式化功能', () => {
  it('应该正确格式化JSON', () => {
    const input = '{"name":"test","value":123}';
    const result = JsonFormatter.format(input, 2);
    const expected = '{\n  "name": "test",\n  "value": 123\n}';
    expect(result).toBe(expected);
  });

  it('应该处理无效JSON', () => {
    const input = '{"invalid":}';
    const result = JsonFormatter.format(input);
    expect(result.success).toBe(false);
    expect(result.error).toContain('JSON parse');
  });

  it('应该支持不同缩进', () => {
    const input = '{"name":"test"}';
    const result2 = JsonFormatter.format(input, 2);
    const result4 = JsonFormatter.format(input, 4);
    expect(result2.data).toContain('  ');
    expect(result4.data).toContain('    ');
  });
});
```

#### 8.1.2 压缩功能测试

```typescript
describe('JsonFormatter 压缩功能', () => {
  it('应该正确压缩JSON', () => {
    const input = '{\n  "name": "test",\n  "value": 123\n}';
    const result = JsonFormatter.compress(input);
    const expected = '{"name":"test","value":123}';
    expect(result).toBe(expected);
  });

  it('压缩后应该移除所有空白字符', () => {
    const input = '{\n  "name"  :  "test"  \n}';
    const result = JsonFormatter.compress(input);
    expect(result).not.toContain('\n');
    expect(result).not.toContain('  ');
  });
});
```

#### 8.1.3 复制功能测试

```typescript
describe('JsonFormatter 复制功能', () => {
  it('应该成功复制到剪贴板', async () => {
    const text = '{"test": "value"}';
    const result = await JsonFormatter.copy(text);
    expect(result.success).toBe(true);
  });

  it('空文本应该返回错误', async () => {
    const result = await JsonFormatter.copy('');
    expect(result.success).toBe(false);
  });
});
```
